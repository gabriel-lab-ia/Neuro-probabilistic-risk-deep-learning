# Responsible Use

This repository is a research prototype for neurological risk stratification and calibrated uncertainty estimation.

- It is not a medical device and does not provide definitive diagnosis.
- Predictions should be treated as decision-support signals that require clinical context.
- Softmax confidence is not the same thing as clinical certainty.
- Predictive uncertainty must be displayed with every model output whenever possible.
- Synthetic data in this MVP is for pipeline validation only and should not be confused with real-world clinical performance.
